Covid-19

Data Visualization

December 31, 2021

Overview: The purpose of this project was to visualize Covid-19 data using the Plotly library in Python. Visualizations of Total Covid Cases by Continent, Total Covid Deaths by Continent, Percent Population Infected and Percent Population Infected over Time were created.

The date range for the data in this project are from 01-01-2020 to 12-08-2021.

Project inspiration can be found here. This project serves as the Python version of that SQL project.

Dataset retrieval was pulled from Our World in Data. Raw Dataset found at JHU CSSE COVID-19 Data.
